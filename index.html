Personal Site: Live | Code

A simple yet elegant grid-based layout with links to all of my work. The HTML is semantic and the site is fully responsive. There is a little bit of JavaScript for a silly excuse generator.

The main problem I ran into was the size of the fonts. Because my design was heavily typographic, I needed to serve three different files which totaled to ~1500kbs—much too large. To solve this, I first converted the .ttf files to .woff2. Then I used pyftsubset to cull the number of glyphs in each file down to only those I needed. After that they totaled just 47kbs, which drastically improved the page's loading speed.

This project was a great way to learn how to use tools such as Lighthosue and Yellow Lab to diagnose and fix performance issues without sacrificing the original design.

Drag-N-Draw: Live | Code

A pixelart sketchpad made with vanilla JavaScript. The pixels are colored in by hovering rather than clicking because I wanted to simulate the effect of an Etch-A-Sketch—that is, you have to be strategic about how you enter and exit the board. The skeuomorphic design evokes a retro arcade game.

This project was primarily an exercise in learning and implementing JavaScript fundamentals such as event handlers and DOM manipulation.
